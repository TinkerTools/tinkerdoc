How to Cite Tinker
------------------

Any published work which utilizes Tinker shall give appropriate acknowledgement to the Tinker 8 developer community's contribution by including Tinker8_.

How to Cite Tinker9
-------------------

In addition to the main Tinker reference Tinker8_, appropriate acknowledgement to the Tinker9 developer community's contribution by including Tinker9_.
